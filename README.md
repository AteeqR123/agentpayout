# Agent Payout Operations Platform

**🚀 Built with Cursor**

## 📦 Libraries & Packages Used

This project was built using a cutting-edge web development stack:
- **React 18** and **Vite 6** for lightning-fast compilation, hot reloading, and robust UI rendering.
- **Tailwind CSS v4** for clean, utility-first styling and dynamic semantic dark/light theme variables.
- **React Router 7** for seamless component routing.
- **Radix UI** primitives for headless, accessible UI components (Tabs, Accordions, Dialogs, etc.).
- **Lucide React** for crisp, scalable vector icons.
- **Motion (Framer Motion)** for elegant layout animations and scroll effects.
- **MUI Core & Emotion** for select enhanced Material UI tools.
- **Recharts** and **React Slick** for dynamic data visualization and interactive carousels.

## 📂 App Pages List

The website is streamlined into the following core pages under the `src/app/pages` module:

- `Login.tsx` - Application authentication and entry.
- `Home.tsx` - Main landing / overview page.
- `AboutUs.tsx` - Company mission, vision, and team overview.
- `Compliance.tsx` - Security and regulatory standards.
- `Contact.tsx` - Form integration for enterprise sales and support.
- `Developers.tsx` - API documentation and sandbox developer resources.
- `Services.tsx` - Complete product feature overviews.
- `MethodLocalBanks.tsx` - Landing page for local bank transfer methods.
- `MethodMobileMoney.tsx` - Landing page for mobile money integrations.
- `MethodUSDT.tsx` - Landing page for crypto (stablecoin) settlement infrastructure.

## ⚙️ How to Run

1. **Install Dependencies:**
   First, install all of the required NPM packages.
   ```bash
   npm install
   ```

2. **Start the Development Server:**
   Launch the development environment to preview the site locally.
   ```bash
   npm run dev
   ```
   *The server defaults to running on http://localhost:5173.*

3. **Build for Production:**
   Bundle your optimized site for deployment.
   ```bash
   npm run build
   ```
